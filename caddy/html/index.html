Portainer shows "API version too old" error:
This is a known issue with Docker 29.x and Portainer. Run these commands to fix:
  • sudo systemctl edit docker.service
  • Add above the "Lines below will be discarded" line:
    [Service]
    Environment=DOCKER_MIN_API_VERSION=1.24
  • Save and exit, then run: sudo systemctl restart docker
  • Finally: docker compose restart portainer
This configures Docker to accept older API versions. See GitHub issue #12925 for updates.
